# 📊 Student Marks Analyzer

## Overview

This project analyzes student marks using Python and NumPy.

It calculates:

- Average marks
- Highest marks
- Lowest marks
- Grade distribution

## Technologies Used

- Python
- NumPy
- Google Colab

## Features

- Accepts marks as input
- Calculates statistics
- Assigns grades
- Uses NumPy arrays

## Learning Outcomes

- Python Programming
- NumPy
- Data Analysis
- Functions
- Loops
- Conditional Statements

## Future Improvements

- Read data from CSV
- Create charts
- Export reports
